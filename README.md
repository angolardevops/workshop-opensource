# Workshop: Git e GitHub - Primeiros Passos para Contribuir com Projetos Open Source

Este repositório contém o material didático, exemplos de código e recursos utilizados no workshop **Git e GitHub: Primeiros Passos para Contribuir com Projetos Open Source**.

## Objetivo

Capacitar iniciantes no uso prático do Git e GitHub, abordando:

- Fundamentos do Git para o dia a dia de desenvolvimento
- Colaboração e versionamento com GitHub
- Estruturação de repositórios bem organizados
- Boas práticas e ética em projetos open source

## Conteúdo

- 📂 `slides/`: apresentação em PDF e PPTX
- 📂 `exemplos/`: comandos e simulações práticas com Git e GitHub CLI
- 📂 `docs/`: materiais de apoio, links e referências

## Público-alvo

Desenvolvedores iniciantes e estudantes interessados em contribuir com software livre.
